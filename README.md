## CLI Chat

A command-line interface for interacting with language models. It has a preset model and system prompt. And other features inside.

**Usage:**

```shell
ChatConsole [options]
```

**Options:**

*   `--endpoint <endpoint_uri>`:  Specifies the URL of the Ollama endpoint.  This option is **optional**.

**Example:**

```shell
ChatConsole --endpoint http://localhost:11434
```
